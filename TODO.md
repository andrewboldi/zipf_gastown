# Zipf's Law Visualization Project

This file outlines the tasks required to build the Zipf's Law visualization tool.

## Phase 1: Backend

- [ ] **Task 1.1: Data Acquisition:** Download a large text file from Project Gutenberg.
- [ ] **Task 1.2: Data Processing:** Create a script to process the text file, count word frequencies, and save the result as a JSON file.
- [ ] **Task 1.3: Backend Server:** Create a simple backend server (Node.js/Express) to serve the processed data.

## Phase 2: Frontend

- [ ] **Task 2.1: React App Setup:** Create a new React application using Create React App.
- [ ] **Task 2.2: API Integration:** Fetch the word frequency data from the backend server.
- [ ] **Task 2.3: Visualization:** Use a charting library (e.g., Chart.js) to visualize the data.
- [ ] **Task 2.4: UI/UX:** Add controls to select different texts and to zoom/pan the chart.

## Phase 3: Integration & Deployment

- [ ] **Task 3.1: Dockerize:** Dockerize the frontend and backend applications.
- [ ] **Task 3.2: Documentation:** Write a README.md with instructions on how to run the project.
